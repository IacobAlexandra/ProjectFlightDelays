Data Science Project - Flight Delay and Cancellation Analysis (2019–2023) 

PHASE 1 

This repository contains the code, data analysis and documentation  developed to explore and predict commercial flight delays using the U.S. Department of Transportation's On-Time Performance dataset from 2019 to 2023

. Using both Python and R, this project covers the complete data analytics lifecycle, including data cleansing, exploratory data analysis (EDA), feature engineering, and statistical hypothesis testing
. The primary modeling objectives include predicting exact arrival delay durations (Regression), categorizing delay severity into distinct tiers (Classification), and identifying operational performance patterns across different airlines and airports (Clustering)
